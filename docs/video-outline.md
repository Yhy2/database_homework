# 校园二手交易平台录屏脚本

## 1. 开场

1. 打开浏览器，输入项目网址。
2. 展示首页，说明系统基于 Vue、Flask 和 MySQL 实现。

## 2. 核心页面浏览

1. 打开「商品列表」页面，展示真实商品数据。
2. 打开「用户列表」页面，展示用户表数据。
3. 打开「订单列表」页面，展示订单和买卖双方信息。

## 3. 数据操作演示

1. 在商品页新增 1 件商品。
2. 修改 1 件未售商品的价格。
3. 删除 1 件未售商品。
4. 刷新页面，证明数据变化已经写入数据库。

## 4. 查询统计演示

1. 打开「查询统计」页面。
2. 展示未售商品、价格大于 30 的商品、生活用品类商品、`u001` 发布的商品。
3. 展示连接查询结果。
4. 展示聚合统计结果。
5. 展示 `sold_item_view` 和 `unsold_item_view`。

## 5. 购买事务演示

1. 打开「购买演示」页面。
2. 选择 1 个买家并购买 1 件未售商品。
3. 跳转到订单页或刷新统计结果，证明订单已写入且商品状态变为已售。
4. 再次尝试购买同一商品，展示重复购买被拒绝。

## 6. 结尾

1. 回到首页，总结系统已覆盖数据库定义、数据操作、查询、视图和事务逻辑。
2. 如已完成公网部署，展示公网网址。
